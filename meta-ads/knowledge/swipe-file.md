## How to Use This File

- Before writing briefs or copy, scan for patterns in your niche
- Look for recurring hook types, copy structures, and visual approaches
- Cross-reference with `trends.md` to see if a swipe pattern aligns with current trends
- Each entry is tagged by niche, funnel stage, hook type, and format
