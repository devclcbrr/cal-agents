# Meta Ads Creative Trends — Knowledge Base

This file is updated by the `/ad-learn` command and the daily scheduled trigger. Each entry includes the source, date learned, and actionable takeaways.

---

## How This Works

1. Run `/ad-learn` with a topic or URL to research and save new findings
2. Run `/ad-learn` with no args to scan curated marketing sources
3. The meta-ads agent reads this file before making creative decisions
4. Entries older than 6 months should be reviewed for staleness
