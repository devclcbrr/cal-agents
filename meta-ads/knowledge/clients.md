# Client Profiles

Brand profiles loaded by the meta-ads agent before producing any creative output. Each profile provides the context needed to write on-brand, audience-aware creatives.

---

## How to Add a Client

When onboarding a new client, use this template:

```
### [Brand Name]

**Business type:** [E-commerce / Service / Hybrid]
**Products/Services:** [What they sell]
**Target audience:**
- Demographics: [Age, gender, location, income]
- Psychographics: [Values, interests, lifestyle]
- Pain points: [Top 3 problems they face]
- Desired outcome: [What they want to achieve]

**Brand voice:** [Tone descriptors — e.g., casual and witty, professional and authoritative]
**Key differentiators:** [What makes this brand different]
**Competitors:** [Top 3 competitors to watch]
**Special ad category:** [None / Housing / Credit / Employment / Political]

**Current offers:**
- [Offer 1]
- [Offer 2]

**Past ad performance:**
- Winners: [What worked and why]
- Losers: [What failed and why]

**Notes:** [Anything else relevant]
```
